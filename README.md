# Product-Management-System-C-
A simple Windows Forms application that creates and stores product information using file handling in C#. The program allows users to generate products, save them to a text file, and load stored data back into the form.

## Features

## 1.Product Creation
User enters product name, brand, and price.
Application generates a random product ID (two letters + four digits).

## 2. Save Products to File
Products are saved to products.txt

## 3.Load Product from File
Clicking Load reads the first line of the file.
The ID, name, brand, and price are displayed back on the form.
